git add . - adicionas todos os arquivos modificados no staging

git log -  mostra o histórico da branch - q para sair e espaço para mostrar a   próxima pagina
git branch - mostra as branchs

git checkout <noma da branch> - troca a branch

git reset - tira arquivo de do staging
git merge <nome da branch> - junta a branch local com outra branch
git rebase - junta a branch, sem criar outo commit
